# ✈️ End-to-End Flight Fare Prediction

## Overview

This project is an end-to-end Machine Learning application that predicts flight ticket prices based on travel details such as airline, source, destination, journey date, departure time, arrival time, duration, and number of stops.

The project demonstrates the complete machine learning lifecycle from data preprocessing and feature engineering to model training, evaluation, experiment tracking, and deployment using a Flask web application.

---

## Key Features

- Flight fare prediction using Machine Learning
- Data preprocessing and feature engineering
- End-to-end ML pipeline implementation
- Model training and evaluation
- Experiment tracking using MLflow
- Data versioning using DVC
- Flask-based prediction interface
